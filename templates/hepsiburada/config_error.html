Hepsiburada API Ayarları Yapılmamış

Hepsiburada entegrasyonunu kullanabilmek için aşağıdaki ortam değişkenlerini yapılandırmanız gerekiyor:

DeğişkenAçıklamaNereden Alınır?
HB_MERCHANT_ID Satıcı ID (GUID) Satıcı Paneli > Bilgilerim
HB_USERNAME API Kullanıcı Adı Entegrasyon ayarları
HB_PASSWORD Servis Anahtarı / Şifre Entegratör Bilgileri > Servis Anahtarı
HB_IS_PRODUCTION Ortam (false=test, true=canlı) İlk başta false yapın
Nasıl yapılır?
  1. Satıcı Paneline giriş yapın
  2. Bilgilerim > Entegrasyon sayfasına gidin
  3. "Entegratör Bilgileri" tabından Servis Anahtarınızı kopyalayın
  4. .env dosyanıza yukarıdaki değişkenleri ekleyin
  5. Uygulamayı yeniden başlatın
