WooCommerce API Yapılandırması Eksik

WooCommerce entegrasyonu için gerekli API ayarları yapılmamış. Lütfen aşağıdaki adımları takip ederek yapılandırmayı tamamlayın.

Yapılandırma Adımları
  1. Projenizin kök dizinindeki .env dosyasını açın
  2. Aşağıdaki değişkenleri ekleyin: 
    WOO_STORE_URL=https://siteniz.com
WOO_CONSUMER_KEY=ck_xxxxxxxxxxxxx
WOO_CONSUMER_SECRET=cs_xxxxxxxxxxxxx
  3. WooCommerce yönetim panelinden API anahtarlarınızı alın:
    • WooCommerce → Ayarlar → Gelişmiş → REST API
    • "Anahtar Ekle" butonuna tıklayın
    • Açıklama girin ve "Okuma/Yazma" yetkisi seçin
    • Consumer Key ve Consumer Secret değerlerini kopyalayın
  4. Uygulamayı yeniden başlatın
